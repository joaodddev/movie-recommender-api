# 🎬 Movie Recommender API

API de recomendação de filmes desenvolvida com **Java + Spring Boot**, consumindo dados da API do TMDB.

## 🚀 Funcionalidades

- 🔍 Buscar filmes por nome
- 🎬 Recomendar filmes similares
- 🌐 Integração com API externa (TMDB)
- 🧠 Arquitetura em camadas (Controller, Service, Client)
- 📦 Uso de DTOs para organização dos dados

## 🛠️ Tecnologias

- Java 17+
- Spring Boot
- WebClient
- REST APIs
